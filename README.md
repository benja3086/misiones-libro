# 📚 Misiones Libro

Este proyecto es una aplicación web creada con **React + Vite** para mostrar y buscar libros de las misiones.

## 🚀 Tecnologías utilizadas

- React
- Vite
- JavaScript
- CSS

## 👀 ¿Qué hace esta app?

- Muestra un listado de libros.
- Tiene un buscador animado.
- Se puede acceder fácilmente a la información de cada libro.
